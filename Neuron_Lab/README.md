## 06: Обучить нейрон для температуры и колличество воды

- Контроль температуры и колличество воды (наличие воды) в чайнике.
- Один нейрон два входа (x1),(x2). 
- x1 - температура воды.
- x2 - скорость нагрева.
Все данные через (scanf). В начале обучаем. Потом в бесконечном цикле (while) вводим данные температуры и скорость нагрева.
